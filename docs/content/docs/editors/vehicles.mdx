## What you’re editing

The Vehicles tab edits entries inside `CVehicleModelInfo__InitDataList/InitDatas`.

## Sections

- **Identity**: modelName, txdName, handlingId, gameName, make
- **Type, Audio & Interaction**: vehicle type/class, layout, driver source extension, and audio hash
- **Visuals**: LOD chip editor, diffuse tint color picker, and dirt range sliders
- **Advanced**: camera presets, wheel/plate/dashboard settings, cover + explosion JSON, first-person POV offsets + IK offsets, trailers/doors/stiffness, VFX/PTFX-style metadata, and vehicle behavior toggles
- **Flags**: grouped toggle list of common GTA vehicle flags

## Linking to handling

`handlingId` should match the `handlingName` from Handling.

If you rename handling, update `handlingId` as well (or you’ll be linking to the wrong physics entry).

## Flags

Flags are stored as a list of string items in XML. METAGEN offers categorized toggles to manage them.
