## What youâ€™re editing

The CarVariations tab edits entries inside `CVehicleModelInfoVariation/variationData`.

## Color sets

CarVariations supports multiple color combinations per model.

Each set includes indices for:

- Primary
- Secondary
- Pearl
- Wheels
- Interior
- Dashboard

## Siren & light settings

These numeric fields can link into CarCols behavior depending on your base game setup and vehicle type.

## Kits

CarVariations includes a list of kit names. The editor lets you add/remove and edit them as strings.

## Windows & plates

- Windows is an integer setting.
- Plate probabilities are edited as three numeric values.
